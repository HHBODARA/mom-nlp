To run this program to get desired output follow the steps and execute it in your terminal:

Step 1: Run python file transcript.py and submit path of audio file and wait for transcription to get complete.

Step 2: After the transcription of audio is completed run test.py file and submit the form with correct details.

Step 3: Run the flask file main.py in local sever. Type below code in your terminal to execute flask file:
Windows:
set FLASK_APP=main.py
flask run

Linux:
export FLASK_APP=main.py
flask run

Step 4: Now enter the url generated by your terminal in local machine web browser.
